# Joshua Waiswa — Profile Page

A smooth, simple React single-page profile site with:

- **Sticky navigation** + smooth scrolling
- **Projects** section
- **Work experience** timeline
- **Hobbies** section
- **Appointment** button/link

## Edit content

Update your details in:

- `src/profileData.ts`

To set your booking URL, change:

- `appointmentLink` in `src/profileData.ts`

## Run locally

```bash
cd profile
npm install
npm run dev
```

## Build

```bash
cd profile
npm run build
npm run preview
```
